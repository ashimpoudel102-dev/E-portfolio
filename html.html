Document Engineering

Semantic HTML5 Markup

Semantic HTML5 defines the structure and meaning of web content, ensuring search engines, screen readers, and developers understand the document architecture.

Lab Laboratory #1

1. Semantic Webpage Layout

Traditional layouts relied heavily on generic <div> elements. Semantic HTML5 introduced structural tags that describe their content explicitly. This aids SEO scrapers and screen readers.

layout.html

    <aside>Sidebar Widget Advertisements</aside>
</main>
<footer>Copyright © Ashim Poudel</footer>
How It Works

<header> identifies navigational headings. <nav> holds menus. <main> isolates primary body resources. <section> breaks up thematic blocks, and <article> encapsulates standalone syndicatable assets.

2. Student Registration Form

HTML forms collect structured input. Using semantic field attributes (like required, specific type fields) ensures standard validation is completed inside the browser prior to backend submission.

form.html
1
2
3
4
5
6
7
8

<form id="student-form">
    <label for="std-name">Name:</label>
    <input type="text" id="std-name" required>
    
    <label for="std-course">Course:</label>
    <select id="std-course">
        <option>CSE</option>
    </select>
</form>
How It Works

<label> connects explicitly to inputs via the for attribute matching the input's id. This increases accessibility. Input attributes like required enable browser-native constraint validation.

Lab Laboratory #3

3. Academic Courses Table

Tables model structured, multi-dimensional relational matrix records. Modern HTML5 uses semantic blocks <thead>, <tbody>, and <tfoot> to frame rows.

table.html
1
2
3
4
5
6
7 
<table>
    <thead>
        <tr><th>Course</th><th>Grade</th></tr>
    </thead>
    <tbody>
        <tr><td>Systems Eng</td><td>Excellent</td></tr>
    </tbody>
</table>
How It Works

<tr> defines table rows. <th> defines header cells (bold, centered by default). <td> represents standard data elements. Structuring tables correctly enables screen readers to parse data row-by-row.

Lab Laboratory #4

4. Responsive Navigation Menus

Websites organize hyperlinks semantically inside unordered lists (<ul>) nested within <nav> blocks. This lets search engine crawlers index site pages efficiently.

nav.html
1
2
3
4
5
6 
<nav>
    <ul>
        <li><a href="index.html">Home</a></li>
        <li><a href="projects.html">Projects</a></li>
    </ul>
</nav>
How It Works

The <ul> (unordered list) tag wraps individual <li> (list item) markers, representing structured menu indexes. Styling libraries then collapse this hierarchy on mobile screens.

5. Multimedia & Responsive Embedding

Embedding audio, video, and map layers is standard in modern layouts. HTML5 features native <video>, <audio>, and <iframe> tags.

media.html
1
2
3 
<iframe 
    src="https://maps.google.com/..." 
    loading="lazy" sandbox></iframe>
How It Works

<iframe> embeds a separate web document inside the parent frame. Adding the loading="lazy" attribute improves initial page loading speed by deferring offscreen iframe loads.
