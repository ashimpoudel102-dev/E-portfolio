Bootstrap 5 Layout Engine

Bootstrap is the world's most popular frontend framework. It features responsive 12-column grid containers, breakpoints, and robust pre-styled UI widgets.

Lab Laboratory #1

1. Bootstrap Responsive Grid System

Bootstrap uses a **12-column layout grid**. Columns scale down automatically on smaller viewports based on breakpoint prefixes (xs, sm, md, lg, xl, xxl).

grid.html
1
2
3
4 
<div class="row">
    <div class="col-12 col-md-6 col-lg-4">Grid Column A</div>
    <div class="col-12 col-md-6 col-lg-8">Grid Column B</div>
</div>
How Grid Works

col-12 forces elements to occupy full width on mobile viewports. On medium viewports, col-md-6 displays them side-by-side. On larger screens, they resize to 4/12 and 8/12 ratios.

2. Responsive SaaS Dashboard Shell

Combining Bootstrap grid alignment utility tokens allows rapid construction of clean dashboard metric dashboards. Custom padding (p-3), cards, and flex margins (ms-auto) handle the layout dynamically.

dashboard.html
1
2

3. Responsive Carousel Slider

The <div class="carousel slide"> element cycles through text and media layers seamlessly. Controls and slide transition animations are managed by Bootstrap's bundled JavaScript engine.

carousel.html
1
2
3
4 
<div class="carousel-item active">
    <div class="carousel-caption d-none d-md-block">
        <h5>First Slide Title</h5>
    </div>
</div>
How Carousel Works

Slide controls trigger JS state updates. The active class dictates which item is visible. The CSS rules apply transition classes like carousel-fade to slide content smoothly.

Lab Laboratory #4

4. Dynamic Overlay Modals

Modals are overlay windows that prioritize key tasks. Bootstrap positions them using absolute layering, dark backdrop filters, and smooth scale-in transitions on open.

modal.html
1
2
3 
<div class="modal fade" id="stdModal">
    <div class="modal-dialog">Modal Content</div>
</div>
How Modals Work

Trigger buttons use attributes like data-bs-toggle="modal" and data-bs-target="#id". The Bootstrap JS library handles showing, centering, and keyboard dismissals automatically.

Lab Laboratory #5

5. Floating Toasts & Alerts

Alerts and Toasts offer non-intrusive notifications. Toasts slide into view (usually in the screen corners) and close automatically after a set delay.

toasts.html
1
2
3 
<div class="toast show" role="alert">
    <div class="toast-body">Toast Notification Msg</div>
</div>
How Toasts Work

Toasts use utility classes like toast, show, and align-items-center. Calling the JavaScript bootstrap.Toast() constructor allows triggering these notifications programmatically.
