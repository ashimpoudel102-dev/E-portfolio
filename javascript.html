5. Real-Time Student Form Validation

Client-side validation verifies data structure prior to submission. This demo performs real-time validation checks for name lengths and email structures using regular expressions.

validate.js
1
2
3
4 
const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
if (!emailRegex.test(valEmail.value)) {
    valEmail.classList.add('is-invalid');
}
How Code Works

Event listeners intercept the form's submit action. The script tests the input values against character constraints and a standard email regular expression pattern (regex), applying styling classes accordingly.
