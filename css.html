Cascading Style Sheets (CSS3) empower developers to convert bare semantic layouts into high-fidelity, responsive, glassmorphic interfaces with micro-animations.

Lab Laboratory #1

1. Flexbox vs. CSS Grid Sandbox

Flexbox is a one-dimensional layout system ideal for distributing space along rows or columns. CSS Grid is a two-dimensional grid system built to handle rows and columns simultaneously.

layout-systems.css
1
2
3
4
5
6 
/* Flexbox Row Configuration */
.flex-layout { display: flex; justify-content: space-between; }

/* CSS Grid 2x2 Grid Configuration */
.grid-layout { display: grid; grid-template-columns: repeat(2, 1fr); gap: 15px; }
How Styling Works

Toggling between flex and grid alters the layout engine. Flex distribution spreads items along the horizontal axis, whereas Grid explicitly aligns boxes into rigid matrix columns.

Lab Laboratory #2

2. Animated Hover Glow Card

Hover interactions provide essential visual feedback. By nesting transitions with box-shadow, transform, and radial gradients, we build highly premium glowing cards.

glow-card.css
1
2
3
4 
.glow-card:hover {
    transform: translateY(-8px) scale(1.02);
    box-shadow: 0 15px 30px rgba(239, 68, 68, 0.3);
}
How Styling Works

Adjusting transform: translateY(-8px) shifts the card upward, simulating elevation. The cubic-bezier transition curves prevent sudden movements, ensuring a premium feel.

Lab Laboratory #3

3. Advanced Buttons & Pseudo-Classes

Buttons are critical interaction targets. Modern style patterns use gradient transitions and absolute-positioned pseudo-elements (::before, ::after) to slide gradients on hover.

buttons.css
1
2
3
4
5 
.glow-btn::after {
    content: ''; position: absolute;
    background: linear-gradient(#ef4444, #f87171);
    opacity: 0; transition: opacity 0.4s ease;
}
How Styling Works

The button contains a secondary gradient background locked inside the ::after pseudo-element at opacity: 0. Hovering shifts its opacity to 1, creating a smooth color transition.

Lab Laboratory #4

4. Glassmorphism Design Elements

Glassmorphism mimics frosted glass overlaying backgrounds. It is defined by multi-layer transparency, semi-opaque borders, and high-intensity backdrop blur filter values.

glass.css
1
2
3 
.glass-container {
    background: rgba(255, 255, 255, 0.05);
    backdrop-filter: blur(12px) saturate(180%);
}
How Styling Works

backdrop-filter: blur(12px) blurs elements behind the glass container, while rgba(255, 255, 255, 0.05) tints it. The fine border anchors it visually.

5. CSS 2D Transition & Transform Sandbox

CSS 2D Transforms allow you to scale, translate, rotate, and skew elements across the horizontal and vertical axes. Combined with transition timers, adjusting these values changes the element's shape dynamically.

transforms.css
1
2
3 
.sandbox-box {
    transition: transform 0.1s ease;
    transform: rotate(15deg) scale(1.1) skewX(5deg);
}
How Styling Works

Adjusting the sliders below changes the element's inline style in real-time, showing how 2D transform matrices calculate rotations and scale expansions dynamically.
